# Conformal-holography-with-curved-light-sheets
Codes for the manuscript "Conformal holography with curved light sheets " 
https://doi.org/10.1364/OE.536859

The file main_senoid_example.m provides the simulation result of Fig. 2(c) and the corresponding CGH to be addressed onto the SLM.
The profile expected to be obtained in the laboratory is also simulated from the CGH.
